TechRecon Accounting System
Cos’è
TechRecon è un gestionale contabile deterministico per aziende italiane, progettato per automatizzare il ciclo documentale–contabile–finanziario garantendo conformità fiscale, tracciabilità e scalabilità controllata.
Il sistema valida, instrada e certifica ogni dato secondo regole contabili e normative esplicite.
Principi chiave
L’anagrafica è la fonte unica di verità.
Nessuna scrittura senza documento.
Controlli a cascata su tutti i flussi.
Errori classificati P0 / P1 / P2.
Funzionalità principali
Ciclo passivo (XML, fornitori, scadenze)
Prima Nota Cassa/Banca separata
Gestione dipendenti e salari con differenze
Riconciliazione bancaria e F24
Corrispettivi contanti/elettronici
Architettura
Backend: FastAPI (Python)
Frontend: React + Vite
Database: MongoDB
Utilizzo
Il sistema è pensato per uso web. Il flusso corretto è:
Configurazione anagrafiche
Import documenti
Controllo alert
Riconciliazione
Report e prima nota
Nota finale
TechRecon è un sistema di controllo contabile: ciò che non è conforme viene bloccato o segnalato.
